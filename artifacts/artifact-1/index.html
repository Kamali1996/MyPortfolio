Artifact 1: Rachel - AI-Powered Health Benefits Assistant

Objective

To build an AI-driven solution that provides accurate answers to common healthcare insurance queries while strictly adhering to critical privacy and medical advice guardrails.

Process

Utilizing the design thinking framework, I identified the core user problem of wait times. I planned the bot's logic, established strict operational boundaries (preventing the intake of PHI), and developed a repository of mock training data. Finally, I executed the build and conducted rigorous QA testing to ensure compliance with established safety instructions.

Tools & Technologies Used

Generative AI (Chatbase), Prompt Engineering, System Role-Prompting, Scenario Testing Methodologies.

Value Proposition

This project demonstrates how generative AI can be securely configured to simplify complex information for end-users, ensuring safety and reliability through structured, boundary-focused testing.
